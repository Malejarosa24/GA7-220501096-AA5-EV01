## Descripción

En esta evidencia se desarrolló una API utilizando Node.js, Express y MongoDB Atlas.

El proyecto permite registrar usuarios e iniciar sesión mediante la validación de credenciales almacenadas en una base de datos MongoDB.

Durante el desarrollo se realizaron pruebas de funcionamiento con Postman y se utilizó GitHub para almacenar y gestionar el código fuente.

## Funcionalidades

- Registro de usuarios.
- Inicio de sesión.
- Conexión con MongoDB Atlas.
- Almacenamiento de información en la base de datos.
- Validación de credenciales.
- Control de versiones con Git y GitHub.

## Estructura del proyecto

- `server.js`: archivo principal del servidor.
- `models/Usuario.js`: modelo de usuario.
- `routes/usuarioRoutes.js`: rutas para registro e inicio de sesión.
- `package.json`: configuración y dependencias del proyecto.

## Evidencias realizadas

- Configuración del proyecto en Visual Studio Code.
- Conexión con MongoDB Atlas.
- Registro de usuarios mediante Postman.
- Inicio de sesión exitoso.
- Validación de errores cuando las credenciales son incorrectas.
- Publicación del proyecto en GitHub.
